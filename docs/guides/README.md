# OpenKotOR Discord Bots — User Guide

This server has three bots. Here is what each one does and where to learn more.

---

## Pazaak Bot

Play pazaak (the card game from KOTOR) against other server members for fake credits.

**Get started:** Type `/pazaak wallet` to see your credits, then `/pazaak challenge` to start a game.

| Command | What it does |
|---|---|
| `/pazaak challenge` | Start a game against someone |
| `/pazaak wallet` | Check your credits and record |
| `/pazaak daily` | Claim 200 free credits (once per day) |
| `/pazaak leaderboard` | See the top 10 players |
| `/pazaak rules` | Read the full game rules |
| `/pazaak rivalry` | See your record against each opponent |

[Full pazaak guide →](pazaak.md)

---

## HK-86 — Role Picker

Pick which topics and pings you want to see on this server. Choose from 13 roles across projects, community, events, and time zones.

**Get started:** Type `/designations panel` to open the role picker and check the boxes you want.

| Command | What it does |
|---|---|
| `/designations panel` | Open the role picker (select multiple at once) |
| `/designations list` | See all available roles |
| `/designations assign` | Add one role |
| `/designations remove` | Remove one role |

[Full HK-86 guide →](https://github.com/OpenKotOR/community-bots/wiki/docs/guides/hk-86)

---

## Trask — Source Lookup

Ask questions about KOTOR modding, tools, and troubleshooting. Trask searches a curated list of trusted community resources and returns matching results.

**Get started:** Type `/ask query:your question here`

| Command | What it does |
|---|---|
| `/ask` | Search for KOTOR resources |
| `/sources` | List all approved sources |

[Full Trask guide →](trask.md)
