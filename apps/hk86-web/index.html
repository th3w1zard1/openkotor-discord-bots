HK-86

Static operator page for the HK-themed reaction-role panels bot. No matchmaking, no backend — configure the bot with JSON on the host and use Discord for everything else.

Add the bot

Pre-filled OAuth link uses the same permission integer as the bot embeds ().

Set VITE_DISCORD_APPLICATION_ID at build time (or GitHub Actions variable HK86_DISCORD_APP_ID) so the invite button can be generated.

Operator checklist

  1. Create panels JSON on the bot host and point HK_REACTION_ROLE_PANELS_PATH at it (hot reload supported).
  2. Run /designations reactions help in Discord for the full setup embed.
  3. Use /designations reactions status (Manage Server) to verify the bot sees your config.
